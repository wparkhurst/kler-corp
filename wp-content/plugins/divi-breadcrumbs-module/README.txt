================== Divi Breadcrumbs Module =================

https://DiviBreadcrumbs.com/
Version: 2.0.0
Author:	CODECRATER
https://divicake.com/author/CODECRATER/

============================================================


This plugin adds a Divi Builder Module which generates breadcrumb navigation menus. Each breadcrumb nav is highly customizable to suit every style imaginable. Brought to you by DiviCake.com



== Installation ==

Installing Divi Breadcrumbs Module can be done by using the following steps:

1. Download the plugin via DiviCake.com, etc.
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
4. There will now be a new “Divi Breadcrumbs” module available to use within the Divi Builder. (Visual builder not yet compatible)


Got questions? visit https://DiviBreadcrumbs.com/ or https://divicake.com/author/CODECRATER/
